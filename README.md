# -Random-Quote-Generator-
проект в GitHub Random Quote Generator (Генератор случайных цитат)  Создать GUI-приложение «Random Quote Generator» с использованием random, сохранением истории и Git.

**Автор:** Галина Головина
## Описание

Графическое приложение для генерации случайных цитат из предопределённого списка. Позволяет сохранять историю сгенерированных цитат и фильтровать её по автору или теме.

## Технологии

- **Python 3**
- **Tkinter** (стандартная библиотека для GUI)
- **JSON** (для хранения данных)
- **random** (для выбора случайной цитаты)
- **Git** (система контроля версий)

## Как запустить

1. Установите Python 3.
2. Клонируйте репозиторий.
3. Запустите приложение:
   ```bash
   python main.py
   ```
4. В главном окне нажмите кнопку «Сгенерировать цитату».
5. Для фильтрации истории введите автора или выберите тему и нажмите «Фильтровать».
6. История сохраняется в файл `history.json` и загружается при запуске.
